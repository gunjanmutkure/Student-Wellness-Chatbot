## Overview
Student Wellness Chatbot is an AI-based application developed as part of the **IBM SkillsBuild Internship**.  
The project focuses on supporting students with **stress management, emotional well-being, and mental wellness** through simple and supportive conversations.

---

## Features
- Exam Mode to support students during exam stress  
- Focus Mode to help improve concentration  
- Vent Mode to allow students to express emotions freely  
- Mood detection based on user input  
- Calm breathing exercises for relaxation  

---

## Technologies Used
- Python  
- Streamlit  
- OpenAI API  
- IBM SkillsBuild Concepts  

---

## Security Note
The OpenAI API key is managed securely using Streamlit secrets and is **not included in the GitHub repository**.

---

## Disclaimer
This project is intended for **educational purposes only** and does not replace professional mental health support.
